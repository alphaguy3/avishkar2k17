About

Softablitz is software developing event. To participate you need to form a team ( max 3 members). A set of problem statements is given to the teams. Each team can choose any of the statements and start developing software for it. The software will be evaluated at the time of Avishkar.

Participation Criteria

Team size : 3

Years participating : B.Tech I,II,III MCA I,II year
